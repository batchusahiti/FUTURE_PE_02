# AI-Powered UGC Ad Content Pack

## Project Overview
This project was created as part of the Prompt Engineering Internship by Future Interns.

The goal of this project is to use AI tools like ChatGPT to create high-converting UGC-style ad content for a local café.

---

## Business Chosen
**Velvet Brews Café** — a cozy café in Pedatadepalli serving students, families, and local coffee lovers.

---

## Features
- UGC-style ad hooks  
- Instagram Reel scripts  
- CTA-focused content  
- AI-generated captions  
- Reusable prompts  
- YouTube Shorts ad scripts  
- Facebook ad copy  

---

## Tools Used
- ChatGPT  
- GitHub  
- Lovable AI (for Task 1 website context)  

---

## Project Structure
- `scripts/` → ad scripts  
- `hooks.txt` → scroll-stopping hooks  
- `captions.txt` → platform captions  
- `prompts.md` → prompts used  
- `README.md` → project overview  

---

## Learning Outcomes
- Improved prompt engineering skills  
- Learned UGC ad content creation  
- Understood AI-based marketing workflows  
- Practiced creating platform-specific ad scripts  

---

## Platforms Covered
- Instagram Reels  
- Facebook Ads  
- YouTube Shorts  

---

## Outcome
This project helped in understanding how prompt engineering and AI tools can be used to create engaging UGC-style marketing content for local businesses like Velvet Brews Café, making social media promotion efficient and creative.
